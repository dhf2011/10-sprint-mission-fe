## 소개
안녕하세요! 판다마켓 프로젝트에 오신 것을 환영합니다! 
판다마켓은 **따뜻한 중고거래를 위한 커뮤니티 플랫폼**이에요.
## 주요 기능 ✨
1. **상품 등록**  
   내가 가진 물건을 올리고, 사진과 설명을 추가해 직접 판매할 수 있어요.
2. **문의 댓글**  
   상품에 대한 궁금한 점이나 의견을 자유롭게 남길 수 있어요. 📝
3. **자유게시판**  
   다양한 주제로 친구들과 이야기를 나누고, 정보를 공유할 수 있는 공간이에요! 🗣️
